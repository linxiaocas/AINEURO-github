## 1.5 Practical Exercises with Solutions / 实用练习及解答

### Exercise 1: Mapping Exercise / 练习1：映射练习

**Problem / 问题**:
Map the following computer components to their neural analogs:
将以下计算机组件映射到它们的神经类比：

1. GPU SIMD units
2. Cache eviction policy
3. Virtual memory page fault
4. Interrupt handler

**Solution / 解答**:

| Component / 组件 | Neural Analog / 神经类比 | Explanation / 解释 |
|----------------|-------------------------|-------------------|
| GPU SIMD units | Gamma-synchronized ensemble / Gamma同步集群 | Parallel processing of similar operations / 相似操作的并行处理 |
| Cache eviction | Synaptic pruning / 突触修剪 | Removal of unused connections / 移除未使用的连接 |
| Page fault | Hippocampal retrieval / 海马检索 | Access to slower storage for missing info / 访问慢速存储获取缺失信息 |
| Interrupt handler | Orienting response / 定向反应 | Rapid shift of attention to salient event / 注意力快速转向显著事件 |

### Exercise 2: Calculation / 练习2：计算

**Problem / 问题**:
Calculate the "neural efficiency" of a processor:
计算一个处理器的"神经效率"：

Given / 给定:
- Power: 65W
- Performance: 100 GFLOPS
- Number of "processing elements": 10⁹ transistors

**Solution / 解答**:

```
1. FLOPS per Watt / 每瓦FLOPS:
   100 GFLOPS / 65W = 1.54 GFLOPS/W

2. FLOPS per transistor / 每晶体管FLOPS:
   100 GFLOPS / 10⁹ = 0.1 FLOPS/transistor

3. Compare to brain / 与大脑比较:
   Brain: 20W, ~10¹⁶ operations/s (estimated synaptic)
   大脑：20瓦，~10¹⁶次操作/秒（估计突触）
   
   Brain efficiency: 5×10¹⁴ ops/W
   大脑效率：5×10¹⁴次操作/瓦
   
   Ratio / 比率: 3×10¹⁴× difference!
   相差3×10¹⁴倍！
```

### Essential Papers / 必读论文

1. **Foundational / 基础**:
   - McCulloch, W.S., & Pitts, W. (1943). A logical calculus of the ideas immanent in nervous activity.
   - von Neumann, J. (1958). The Computer and the Brain.

2. **Modern Integration / 现代综合**:
   - Hassabis, D., et al. (2017). Neuroscience-inspired artificial intelligence.
   - Marblestone, A.H., Wayne, G., & Kording, K.P. (2016). Toward an integration of deep learning and neuroscience.

3. **Consciousness / 意识**:
   - Tononi, G., et al. (2016). Integrated information theory: From consciousness to its physical substrate.
   - Dehaene, S., & Changeux, J.P. (2011). Experimental and theoretical approaches to conscious processing.

### Online Resources / 在线资源

- **AINEURO GitHub**: https://github.com/linxiaocas/AINEURO-github/
- **Neuroscience Databases**: NeuroData, Allen Brain Atlas
- **Simulation Tools**: NEST, NEURON, Brian

### Courses / 课程

- **Computational Neuroscience**: Coursera (University of Washington)
- **Deep Learning Specialization**: Coursera (DeepLearning.AI)
- **Computer Architecture**: MIT OpenCourseWare
